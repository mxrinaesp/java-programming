## First Java file
    - debe haber una <strong>class</strong> que coincida con el nombre del archivo

    - dentro de esa class va lo principal de los archivos java: <em>main method</em>:
        <strong>public static void <em>main</em> (String args[]){}</strong>

        * void: no necesita return un valor
        * static: podemos ejecutar el method sin necesidad de un objeto

    -dentro del method escribimos lo que va a aparecer(print) en el output cuando "run code":
        <strong>System.out.print("...");</strong>

    -ctrl-s y guardar con la extensión .java
